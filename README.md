# 📖 OOP School Library <a name="about-project"></a>

**OOP School Library** is an app that records what books are in the library and who borrows them. It allows the user to add new students or teachers, add new books and save records of who borrowed a given book and when it was borrowed..

## 🛠 Built With <a name="built-with"></a>

- Ruby

### Tech Stack <a name="tech-stack"></a>

### Key Features <a name="key-features"></a>
